# Hello Godot

It's a time honored tradition amongst programmers to begin their journey through a new *programming environment* *(in our case: the [Godot game engine / editor](https://godotgameengine.org))* with a *"hello world" project*.

A classic tool in any developer's toolkit, the *"hello world" project* allows us to make a very simple program, just to see if we can get it to run.
